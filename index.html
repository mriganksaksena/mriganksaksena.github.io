I'm currently developing a trading algorithm that uses a gernalized autoregressive conditional heteroskedastic model (GARCH) to forecast volatiliy ot specfic securities. The algorithm parametricizes the GARCH portion through a maximum likelihood estimation (MLE) using brownian motion. Based on various momentum algorithms, the volatility is used to price point a stock and develop a score. The scores are ranked according to a Spearman's ranking coefficient and traded along with safety stocks based using a weighted average system. I'm currently working on implementing beta hedging and reducing max drawdown in order to stabilize returns.
